# 🍽️ グルメ記録 - お店と写真の思い出

訪れたお店と料理の写真を記録するPWA（Progressive Web App）です。スマホのホーム画面に追加してアプリのように使えます。

## ✨ 主な機能

- 📸 **複数写真対応**: 1つの記録に複数の写真を保存可能
- 📅 **訪問日記録**: いつ訪れたかを記録
- 💬 **コメント機能**: 料理の感想やお店の雰囲気をメモ
- 🔍 **検索機能**: お店の名前で素早く検索
- 💾 **データ管理**: JSONファイルでエクスポート/インポート
- 📱 **オフライン対応**: インターネット接続なしでも動作
- 🎨 **レスポンシブデザイン**: スマホ・タブレット・PCで快適に使用可能

## 📱 スマホでの使い方

### 1. アプリにアクセス

デプロイされたURLにスマホのブラウザ（Safari、Chrome等）でアクセスします。

### 2. ホーム画面に追加

#### iPhoneの場合:
1. Safariで開く
2. 画面下部の「共有」ボタン（□に↑のアイコン）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

#### Androidの場合:
1. Chromeで開く
2. 画面右上のメニュー（⋮）をタップ
3. 「ホーム画面に追加」または「アプリをインストール」を選択
4. 「追加」をタップ

### 3. アプリとして使用

ホーム画面のアイコンをタップすると、アプリのように全画面で起動します！

## 🚀 GitHub Pagesへのデプロイ方法

### 手順1: GitHubリポジトリの作成

1. [GitHub](https://github.com)にログイン
2. 右上の「+」→「New repository」をクリック
3. リポジトリ名を入力（例: `gourmet-log`）
4. 「Public」を選択
5. 「Create repository」をクリック

### 手順2: ファイルのアップロード

1. 作成したリポジトリのページで「uploading an existing file」をクリック
2. 以下のファイルをドラッグ&ドロップ:
   - `index.html`
   - `style.css`
   - `script.js`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
   - `README.md`
3. 「Commit changes」をクリック

### 手順3: GitHub Pagesの有効化

1. リポジトリの「Settings」タブをクリック
2. 左サイドバーの「Pages」をクリック
3. 「Source」で「Deploy from a branch」を選択
4. 「Branch」で「main」を選択、フォルダは「/ (root)」を選択
5. 「Save」をクリック

### 手順4: アクセス

数分後、以下のURLでアクセス可能になります:
```
https://[あなたのユーザー名].github.io/[リポジトリ名]/
```

例: `https://username.github.io/gourmet-log/`

## 💡 使い方のヒント

- **写真は複数選択可能**: ドラッグ&ドロップまたはファイル選択で複数枚追加できます
- **データのバックアップ**: 定期的に「保存」ボタンでJSONファイルをダウンロードしましょう
- **機種変更時**: エクスポートしたJSONファイルを新しい端末で読み込めば、データを移行できます
- **オフラインでも使える**: 一度アクセスすれば、ネット接続なしでも記録の閲覧・追加が可能です
